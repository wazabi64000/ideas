Étape 1 – Initiation du projet

Dans cette étape, nous allons préparer le projet pour le développement. Nous allons créer l’arborescence de base, initialiser Node.js et installer les dépendances principales. Cela garantit un projet structuré et prêt pour la suite.

Dépendances et outils

  • Node.js & npm : pour gérer le projet et installer les packages
  • Express : framework web pour créer le serveur
  • dotenv : pour gérer les variables d’environnement
  • nodemon : pour relancer automatiquement le serveur lors des modifications
  • mysql2 : pour interagir avec la base de données MySQL
  • Joi : pour la validation des données
  • Twig : moteur de templates côté serveur
  • bcrypt, jsonwebtoken : pour l’authentification et la sécurité
  • Multer : pour l’upload des images

Théorie et code

L’objectif ici est de créer la structure initiale et d’installer les dépendances. Voici un exemple de commandes et d’arborescence : 


# Initialisation du projet
npm init -y

# Installation des dépendances principales
npm install express dotenv nodemon mysql2 joi twig bcrypt jsonwebtoken multer body-parser

# Arborescence proposée
boite-a-idees/
├─ config/
│  └─ db.js
├─ controllers/
├─ middleware/
├─ models/
├─ routes/
├─ views/
│  ├─ layout.twig
│  ├─ idees.twig
│  └─ nouvelle-idee.twig
├─ public/
│  ├─ css/styles.css
│  └─ js/script.js
├─ uploads/
├─ .env
└─ server.js

Cette structure permet de séparer clairement la configuration, la logique métier, les routes, les modèles, les vues et les fichiers statiques.

Commandes et exécution


# Lancer le serveur en développement
npx nodemon server.js

# Vérifier que le projet démarre correctement
node server.js

Récapitulatif

  • Projet Node.js initialisé
  • Dépendances principales installées
  • Arborescence du projet définie
  • Projet prêt pour la configuration de la base de données
