Bienvenue dans le projet « Boîte à Idées »

Ce guide complet vous accompagne pas à pas dans la création d'une application web fullstack moderne avec :

  • Node.js et Express ESM côté serveur
  • Base de données MySQL
  • Template Twig pour le rendu dynamique
  • Front-end responsive et accessible
  • CRUD complet, authentification et gestion des rôles

Le projet sera développé progressivement au fil des étapes, chaque étape ajoutant une fonctionnalité ou améliorant le code existant.
