Étape 8 – Affichage interactif des idées côté client

Nous allons créer des cartes pour afficher les idées de façon dynamique et responsive, en utilisant Twig pour récupérer les données depuis le serveur.


// idees.twig – affichage des idées
{% for idee in idees %}


  
{{ idee.titre }}

  
{{ idee.description }}

  {% if idee.image %}
    
  {% endif %}
  Créée par {{ idee.createur }} le {{ idee.dateCreation | date("d/m/Y") }}


{% endfor %}
